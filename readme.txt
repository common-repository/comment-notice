The main function of this plugin is to display a subscription option on the end of the comment area, 
so they can be noticed when there are new comments on this post.

== Description ==

**Main Features:**

1. List comment subscriptions email, post name, date.
2. Ban or Active the subscribers.
3. Modify email notification subject, body, footer by tags.
4. Modify email styles, include body background, text color and footer background.
5. Change the display text on the end of comment area.


**More info:**

* [Leave a comment](http://plugins.wp-coder.net/purchase/comment-notice/)

== Installation ==

1. Upload the contents of comment-notice.zip to your plugins directory.
2. Activate the plugin
3. There will be a "Notice" menu on your admin panel. 
4. Set email contant and style on the "Email format" menu.
